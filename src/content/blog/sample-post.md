## Getting Started

Create new posts by adding `.md` files to the `src/content/blog/` directory.

### Frontmatter Fields

| Field | Required | Description |
| --- | --- | --- |
| `title` | Yes | Post title |
| `description` | Yes | Post excerpt for SEO |
| `pubDatetime` | Yes | Publication date (ISO 8601) |
| `author` | No  | Defaults to site author |
| `featured` | No  | Show on homepage featured section |
| `draft` | No  | Set `true` to hide from production |
| `tags` | No  | Array of tag strings |
| `ogImage` | No  | Custom OG image path |

Delete this file and start creating your own content!
